# Welcome to mdr

A **minimal** markdown reader for the terminal.

## Features

- Syntax colored headings
- **Bold** and *italic* text
- `Inline code` highlighting
- Code blocks
- Lists (ordered and unordered)
- Horizontal rules
- Block quotes

## Code Example

```rust
fn main() {
    println!("Hello from mdr!");
}
```

### Keybindings

- `j` / `Down` — scroll down
- `k` / `Up` — scroll up
- `d` / `PageDown` — half page down
- `u` / `PageUp` — half page up
- `g` / `Home` — go to top
- `G` / `End` — go to bottom
- `Space` — page down
- `q` / `Esc` — quit

That's it. ***Minimal***.
